EXO-NET

Pro v2.0.0

Advanced network reconnaissance and port scanning toolkit with professional pentest reporting. CVE detection, risk scoring, and stealth profiles built in.

v2.0.0 Highlights

  • Interactive menu — no commands needed
  • Live TUI dashboard with rich UI
  • CVE vulnerability detection
  • Risk scoring engine (0–100)
  • Professional HTML pentest reports

Capabilities

  • Host discovery (ARP + ICMP + TCP)
  • TCP/SYN/UDP port scanning
  • Service banner grabbing
  • OS fingerprinting
  • CVE vulnerability detection
  • Risk scoring engine
  • Professional HTML reports
  • T0–T5 timing profiles
  • Evasion mode

EXO-OSINT

v2.0.0

Professional Open Source Intelligence framework for IP, domain, email, username and phone investigation. Correlation engine, footprint scoring, and interactive map-driven reports.

v2.0.0 Highlights

  • Interactive menu — no commands needed
  • 85+ platform username hunter
  • Correlation engine — finds linked accounts
  • Digital footprint score (0–100)
  • Live investigation dashboard
  • Interactive HTML reports with world map

Capabilities

  • IP geolocation + ASN + abuse check
  • Domain WHOIS + DNS + SSL analysis
  • Email breach + reputation check
  • Username hunting (85+ platforms)
  • Phone number intelligence
  • Correlation engine
  • Digital footprint scoring
  • Interactive HTML reports with maps
