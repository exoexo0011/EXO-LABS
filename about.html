// Mission

Why EXO LABS exists

Building professional-grade open-source security tools for ethical hackers and security researchers.

The cybersecurity community runs on the shoulders of free, auditable, well-engineered tools. EXO LABS contributes to that ecosystem by shipping focused, no-bloat reconnaissance and intelligence frameworks that are practical for real engagements and accessible to learners.

Every tool is MIT-licensed, written in clean Python, documented, and designed to integrate into a modern blue-team or red-team workflow.

// Timeline

Project Timeline

  1. v0 // Early 2025

    Concept & first prototypes

    Initial Python prototypes for host discovery and DNS enumeration. Internal-only, hand-rolled scripts.

  2. v1 // Mid 2025

    EXO-NET goes public

    First open-source release of EXO-NET. ARP/ICMP/TCP discovery, port scan, banner grab and basic HTML report.

  3. v2 // Late 2025

    EXO-OSINT framework released

    Full OSINT framework: IP, domain, email, username and phone correlation, with map-driven reports.

  4. v2.0.0 // 2026

    Pro release: CVE engine + risk scoring

    EXO-NET Pro v2.0.0 ships with CVE detection, risk scoring, T0–T5 timing profiles and evasion mode. EXO-OSINT v2.0.0 adds the correlation engine and digital footprint scoring.
